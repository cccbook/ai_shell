# 10. Разбор аргументов и CLI инструменты

---

## 10.1 Основы аргументов командной строки

```bash
#!/bin/bash

echo "Скрипт: $0"
echo "Первый аргумент: $1"
echo "Второй аргумент: $2"
echo "Третий аргумент: ${3:-по умолчанию}"  # значение по умолчанию
echo "Количество аргументов: $#"
echo "Все аргументы: $@"
```

### Выполнение

```bash
./script.sh foo bar
# Вывод:
# Скрипт: ./script.sh
# Первый аргумент: foo
# Второй аргумент: bar
# Третий аргумент: по умолчанию
# Количество аргументов: 2
# Все аргументы: foo bar
```

## 10.2 Простой разбор аргументов

### Позиционные параметры

```bash
#!/bin/bash

if [[ $# -eq 0 ]]; then
    echo "Использование: $0 <файл>"
    exit 1
fi

FILE="$1"

if [[ ! -f "$FILE" ]]; then
    echo "Ошибка: файл не существует"
    exit 1
fi

echo "Обработка $FILE..."
```

### Обработка необязательных аргументов

```bash
#!/bin/bash

VERBOSE=false
OUTPUT=""

while [[ $# -gt 0 ]]; do
    case "$1" in
        -v|--verbose)
            VERBOSE=true
            shift
            ;;
        -o|--output)
            OUTPUT="$2"
            shift 2
            ;;
        -*)
            echo "Неизвестная опция: $1"
            exit 1
            ;;
        *)
            FILE="$1"
            shift
            ;;
    esac
done

$VERBOSE && echo "подробный режим включён"
[[ -n "$OUTPUT" ]] && echo "Вывод в: $OUTPUT"
```

## 10.3 `getopts`: Стандартный разбор опций

```bash
#!/bin/bash

while getopts "hv:o:" opt; do
    case $opt in
        h)
            echo "Справка"
            exit 0
            ;;
        v)
            echo "подробный режим: $OPTARG"
            ;;
        o)
            echo "выходной файл: $OPTARG"
            ;;
        \?)
            echo "Неверная опция: -$OPTARG" >&2
            exit 1
            ;;
    esac
done

shift $((OPTIND - 1))

echo "Оставшиеся аргументы: $@"
```

### Справочник форматов опций

| Формат | Значение |
|--------|----------|
| `getopts "hv:"` | `-h` без аргумента, `-v` требует аргумент |
| `OPTARG` | Значение аргумента текущей опции |
| `OPTIND` | Индекс следующего аргумента |

## 10.4 Интерактивный ввод

### `read`: Чтение пользовательского ввода

```bash
#!/bin/bash

# Базовое чтение
read -p "Введите ваше имя: " name
echo "Привет, $name"

# Пароль (скрытый ввод)
read -sp "Введите пароль: " password
echo

# Чтение нескольких значений
read -p "Введите имя и возраст: " name age
echo "$name, $age лет"

# Таймаут
read -t 5 -p "Введите в течение 5 секунд: " value
```

### Подтверждение

```bash
confirm() {
    read -p "$1 (д/н) " -n 1 -r
    echo
    [[ $REPLY =~ ^[Дд]$ ]]
}

if confirm "Удалить этот файл?"; then
    echo "Удаление..."
fi
```

## 10.5 Интерфейс меню

```bash
#!/bin/bash

PS3="Выберите операцию: "

select choice in "Создать проект" "Удалить проект" "Выход"; do
    case $choice in
        "Создать проект")
            echo "Создание..."
            ;;
        "Удалить проект")
            echo "Удаление..."
            ;;
        "Выход")
            echo "До свидания!"
            exit 0
            ;;
        *)
            echo "Неверный выбор"
            ;;
    esac
done
```

## 10.6 Краткий справочник

| Синтаксис | Описание |
|-----------|----------|
| `$0` | Имя скрипта |
| `$1`, `$2`... | Позиционные параметры |
| `$#` | Количество аргументов |
| `${var:-default}` | Значение по умолчанию |
| `getopts "hv:" opt` | Разбор опций |
| `$OPTARG` | Аргумент текущей опции |
| `read -p "подсказка:" var` | Чтение ввода |
| `read -s var` | Скрытый ввод (пароль) |
| `read -t 5 var` | Таймаут 5 секунд |
| `select` | Интерфейс меню |

## 10.7 Упражнения

1. Напишите CLI инструмент, принимающий `-n` для количества, `-v` для подробного режима
2. Используйте `getopts` для разбора опций `-h` (справка), `-o` (выходной файл)
3. Напишите функцию подтверждения, которая продолжает только при ответе "д"
4. Создайте простое меню калькулятора с помощью `select`
