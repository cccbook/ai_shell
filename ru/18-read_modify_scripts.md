# 18. Чтение и изменение скриптов других авторов

---

## 18.1 Зачем читать чужие скрипты

- Принятие проекта на сопровождение
- Использование инструментов с открытым исходным кодом
- Отладка проблем
- Изучение новых техник

ИИ ежедневно сталкивается с незнакомыми скриптами, поэтому умение быстро понимать Shell-скрипты других авторов необходимо.

---

## 18.2 Начальное исследование

### Шаг 1: Проверьте shebang

```bash
head -1 script.sh
```

```bash
#!/bin/bash      # Использовать bash
#!/bin/sh        # Использовать POSIX sh (более совместимо)
```

### Шаг 2: Проверьте права и размер

```bash
ls -la script.sh
wc -l script.sh
```

### Шаг 3: Быстрая проверка синтаксиса

```bash
bash -n script.sh  # Проверить только синтаксис, не выполнять
```

---

## 18.3 Понимание структуры

### Типичная структура

```bash
#!/bin/bash
# Комментарий: описание скрипта

# Настройка
set -euo pipefail
VAR="value"

# Определения функций
function help() { ... }

# Основной поток
main() { ... }

# Выполнение
main "$@"
```

### Найдите основной поток

```bash
# Проверить последние строки
tail -20 script.sh

# Найти определения функций
grep -n "^function\|^[[:space:]]*[a-z_]*\(" script.sh
```

---

## 18.4 Команды для анализа

```bash
# Найти все определения функций
grep -n "^[[:space:]]*function" script.sh

# Найти все условные операторы
grep -n "if\|\[\[" script.sh

# Найти все циклы
grep -n "for\|while\|do\|done" script.sh

# Найти подстановки команд
grep -n '\$(' script.sh

# Найти все выходы
grep -n "exit" script.sh
```

---

## 18.5 Проверка безопасности

```bash
# Найти опасные команды
grep -n "rm -rf" script.sh
grep -n "sudo" script.sh
grep -n "eval" script.sh

# Проверить риски инъекции переменных
grep -n '\$[A-Za-z_][A-Za-z0-9_]*[^"}]' script.sh
```

---

## 18.6 Упражнения

1. Проанализируйте существующий Shell-скрипт с помощью `grep` и `awk`
2. Найдите все переменные в скрипте и поймите их назначение
3. Используйте `bash -n` для проверки синтаксиса скрипта
4. Добавьте комментарии к скрипту без комментариев, объясняя каждый раздел
